Python 3.6.5
Install pandas,flask,xlrd libraries in python
(e.g.
python pip -m install flask
python pip -m install pandas
python pip -m install xlrd
)

All files need to be under the upload folder and of type excel in order type .xls and .xlsx

Open cmd prompt
->navigate to reportApp folder
->run the python flask server : py main.py
->open chrome browser :http://localhost:5000/

Document Structure
The main folder has the py files
The static folder has css, javascript files and images
The templates folder had the html files

Note: Big excel files with large number of columns take time to load. 
Wait for sometime when you press selectall option in that case.
